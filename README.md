# 🔐 Next.js Authentication with Supabase


A simple Next.js app with user authentication using **Supabase**. This project includes:

- ✅ Sign Up Page  
- 🔑 Login Page  
- 🏠 Protected Home Page  
- 🚪 Logout Functionality  
- ⚡ Supabase Auth (Email & Password)
